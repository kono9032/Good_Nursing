# Good_Nursing
간호사의 업무 부담을 줄여주는 결합형 체온-혈압계
- 주요 내용 : 체온계와 혈압계가 결합된 구조로 휴대가 편리한 Iot 의료기기
- 주요 업무 : HW, SW 제작 일체
- 기술 스택 : Arduino, RaspberryPI, HTML, SoildWorks, C, python, Mysql
- 업무 기간 : 2021.12~2022.10 (약 10개월)
